Damped Oscillator

A second-order damped oscillator driven by an external input \(u(t)\):

\[\ddot{x} + 2\zeta\omega_0\,\dot{x} + \omega_0^2\,x = u(t)\]

Drag the Input (x) slider to apply a force \(u(t)\); adjust \(\zeta\) (damping ratio) and \(\omega_0\) (natural frequency) to change the dynamics. At \(\zeta = 0\) the system oscillates forever; at \(\zeta = 1\) it is critically damped; above that it is overdamped.
